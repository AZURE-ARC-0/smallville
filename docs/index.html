What Are Generative Agents?

Generative agents are virtual characters that react to their environment using stored memories. Using LLM models such as ChatGPT, agents are able to observe their surroundings, store memories, and react to state changes in the world

Generative Agents save time programming individual interactions by hand and make NPC's more realistic
